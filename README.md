# CS273P Final Project

## Overview
Final project for CS273P — Machine Learning. Details on the problem, model architecture, and results will be added as the project develops.

## Project Requirements
- **Objective:** Design, implement, and solve a real-world machine learning problem using deep learning in PyTorch.
- **Technical Depth:** Must go beyond off-the-shelf sklearn models or basic decision trees. Expected to include custom architectures, loss functions, or training strategies with thoughtful experimentation and ablation studies.
- **Encouraged Techniques:** Neural nets, CNNs, RNNs, transformers, deep generative models, graph neural nets, LLMs.
- **Open-Source Usage:** Pretrained models and open-source code are allowed, but all borrowed code must be documented and meaningful original contribution is required.
- **Deliverables:**
  - Final written report (problem definition, related work, dataset, model design, training, evaluation, results, analysis, limitations)
  - GitHub repository with README, setup instructions, reproducible training/evaluation scripts, and a demo Jupyter notebook
- **Team Size:** Up to 3 members.

## Project Structure
